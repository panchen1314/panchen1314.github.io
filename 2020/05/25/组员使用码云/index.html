组员使用码云

发表于 2020-05-25

1、加入团队

前提是已经注册好了码云。https://gitee.com/ 进行注册登录

点击下边链接，同意加入团队

2、克隆公共部分的代码

方式一：git clone https://gitee.com/panchen1808935480/c361.git

方式二：直接点击下载zip

3 初次使用git设置签名

cmd进入终端

4 初次生成公钥

回车 y

点击码云右上角头像 / 设置 / SSH公钥 粘贴生成的的公钥

5 利用 Vscode 插件 更新仓库

5.1、 新建一个分支，建议使用自己的名字，比如：panchen

5.2、切换到 panchen分支

5.3、提交到本地

当你进行了代码修改之后，点击“source control”按钮

先点击 “+” 号，进入到暂存的更改，添加此次修改的备注，点击确认

​

5.4、提交到远程仓库

6.进入码云查看

进入刚加入的此项目，点击查看分支，检查是否有自己刚刚创建的分支，如果有自己的分支，说明推送成功，如果没有，则需要检查自己是否推送成功。

7.创建合并请求

7.1在当前项目，点击pull request，新建pull Request

7.2将自己的分支合并到dev分支，写上标题，注释，其他选项不用勾选，直接点击创建

7.3手动合并

​ 因为当前创建的Pull Request不能自动合并，所以就需要手动合并
